# 딥시그널 - 구글 시트 연동 가이드

## 📋 준비물
- 구글 계정
- 5분의 시간

---

## 🚀 설정 방법 (단계별)

### 1단계: 구글 시트 생성

1. [Google Sheets](https://sheets.google.com) 접속
2. **"새로 만들기"** 클릭하여 빈 스프레드시트 생성
3. 시트 이름을 **"딥시그널 신청서"**로 변경 (선택사항)

---

### 2단계: 시트 설정

1. 첫 번째 행에 다음 헤더를 입력:

   | A열 | B열 | C열 | D열 |
   |-----|-----|-----|-----|
   | **신청일시** | **이름** | **연락처** | **접속경로** |

2. 헤더 행을 선택하고 **굵게** 만들기 (선택사항)

---

### 3단계: Apps Script 설정

1. 구글 시트 상단 메뉴에서 **확장 프로그램 > Apps Script** 클릭

2. 기본 코드를 모두 삭제

3. `google-apps-script.js` 파일의 내용을 전체 복사하여 붙여넣기

4. 상단에서 프로젝트 이름을 **"딥시그널 신청서 API"**로 변경 (선택사항)

5. **저장** 버튼 클릭 (💾 아이콘)

---

### 4단계: 웹 앱 배포

1. 상단 우측의 **배포 > 새 배포** 클릭

2. 설정 입력:
   - **유형 선택**: ⚙️ 아이콘 클릭 → **웹 앱** 선택
   - **설명**: "딥시그널 신청서 v1" (선택사항)
   - **실행 계정**: **나**
   - **액세스 권한**: **모든 사용자** ⚠️ 중요!

3. **배포** 버튼 클릭

4. 권한 승인:
   - "Google에서 확인하지 않은 앱" 경고 → **고급** 클릭
   - **"[프로젝트 이름](안전하지 않은 페이지)로 이동"** 클릭
   - **허용** 버튼 클릭

5. **웹 앱 URL 복사** 📋
   ```
   예시: https://script.google.com/macros/s/AKfycbz.../exec
   ```

---

### 5단계: 랜딩페이지에 URL 적용

1. `index.html` 파일 열기

2. 1446번째 줄 근처에서 다음 코드 찾기:
   ```javascript
   const GOOGLE_SCRIPT_URL = 'YOUR_GOOGLE_SCRIPT_URL_HERE';
   ```

3. 따옴표 안의 내용을 복사한 URL로 교체:
   ```javascript
   const GOOGLE_SCRIPT_URL = 'https://script.google.com/macros/s/AKfycbz.../exec';
   ```

4. 파일 저장

---

## ✅ 테스트하기

1. `index.html` 파일을 브라우저로 열기
2. **"무료 체험하기"** 버튼 클릭
3. 테스트 정보 입력:
   - 이름: 홍길동
   - 연락처: 010-1234-5678
4. **"신청 완료"** 클릭
5. 구글 시트로 돌아가서 데이터가 추가되었는지 확인

## 🔧 문제 해결

### 문제: "신청 중 오류가 발생했습니다" 메시지 표시
**해결책:**
- Apps Script 배포 시 **"액세스 권한"을 "모든 사용자"로** 설정했는지 확인
- URL이 정확히 복사되었는지 확인 (끝에 `/exec`가 있어야 함)

### 문제: 데이터가 시트에 저장되지 않음
**해결책:**
- 시트 이름이 **"신청서"**인지 확인 (또는 Apps Script의 SHEET_NAME 수정)
- Apps Script에서 **저장**을 눌렀는지 확인
- 웹 앱을 **새로 배포**해보기 (배포 > 배포 관리 > 새 버전으로 배포)

### 문제: 권한 오류
**해결책:**
- Apps Script 편집 화면에서 실행 > doPost 함수 실행
- 권한 승인 다시 진행

## 📱 알림 설정 (선택사항)

신청이 들어올 때마다 이메일로 알림받기:

1. 구글 시트에서 **도구 > 알림 규칙** 클릭
2. **"다른 사용자가 수정하면"** 선택
3. **"즉시 이메일 전송"** 선택
4. **저장**
